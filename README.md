# 42 Mobile Piscine

Introduction to mobile development as part of the course at [42 School Berlin](https://42berlin.de/).

All exercises and apps in this repository are built with **[Flutter](https://flutter.dev/)** — Google’s UI toolkit for compiling to mobile, web, and desktop from a single [Dart](https://dart.dev/) codebase.

Projects are grouped by module. Each app has its own README with setup and feature notes.
